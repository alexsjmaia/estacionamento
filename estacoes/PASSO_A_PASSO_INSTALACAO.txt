PASSO A PASSO PARA INSTALAR O AGENTE DE IMPRESSAO LOCAL NO WINDOWS

OBJETIVO
ESTE AGENTE FAZ A IMPRESSAO SAIR DIRETAMENTE NA IMPRESSORA DO COMPUTADOR LOCAL
ONDE O SISTEMA FOI ABERTO NO NAVEGADOR.

ARQUIVOS DESTA PASTA
1. LOCAL_PRINT_AGENT.py
2. START_PRINT_AGENT.bat
3. PASSO_A_PASSO_INSTALACAO.txt

REQUISITOS NO COMPUTADOR DA ESTACAO
1. WINDOWS
2. PYTHON INSTALADO
3. PYWIN32 INSTALADO
4. IMPRESSORA ELGIN INSTALADA NESTE COMPUTADOR
5. NOME DA IMPRESSORA IGUAL A Elgin

COMO INSTALAR
1. COPIE A PASTA `estacoes` PARA O COMPUTADOR QUE VAI IMPRIMIR.
2. ABRA O POWERSHELL DENTRO DA PASTA COPIADA.
3. INSTALE O PYWIN32:
   pip install pywin32
4. TESTE A IMPRESSORA NO WINDOWS E CONFIRME O NOME:
   Elgin

COMO TESTAR O AGENTE
1. EXECUTE:
   python LOCAL_PRINT_AGENT.py
2. NO NAVEGADOR DA MESMA ESTACAO, TESTE:
   http://127.0.0.1:8951/health
3. PARA LISTAR IMPRESSORAS:
   http://127.0.0.1:8951/printers

COMO RODAR EM MODO OCULTO
1. CRIE UMA PASTA FIXA:
   C:\ESTACIONAMENTO\ESTACOES
2. COPIE OS ARQUIVOS PARA ESSA PASTA.
3. TESTE:
   pythonw C:\ESTACIONAMENTO\ESTACOES\LOCAL_PRINT_AGENT.py

COMO INICIAR AUTOMATICAMENTE COM O WINDOWS
1. PRESSIONE:
   WIN + R
2. DIGITE:
   shell:startup
3. CRIE O ARQUIVO:
   INICIAR_PRINT_AGENT_OCULTO.vbs
4. COLE O CONTEUDO:

Set WshShell = CreateObject("WScript.Shell")
WshShell.Run "pythonw C:\ESTACIONAMENTO\ESTACOES\LOCAL_PRINT_AGENT.py", 0, False

5. SALVE O ARQUIVO.

OBSERVACOES IMPORTANTES
1. O AGENTE PRECISA FICAR ABERTO NO COMPUTADOR LOCAL.
2. O SISTEMA WEB SO VAI IMPRIMIR NESTA ESTACAO SE O AGENTE ESTIVER RODANDO.
3. O SISTEMA USA:
   http://127.0.0.1:8951/print-base64
4. A SAIDA DE VEICULOS IMPRIME 2 VIAS COM INTERVALO DE 5 SEGUNDOS.

SE DER ERRO
1. VERIFIQUE SE O PYTHON ESTA INSTALADO.
2. VERIFIQUE SE O `pip install pywin32` FOI EXECUTADO.
3. VERIFIQUE SE A IMPRESSORA ESTA INSTALADA.
4. VERIFIQUE SE O NOME DA IMPRESSORA ESTA COMO `Elgin`.
5. TESTE:
   http://127.0.0.1:8951/health
